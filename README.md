# Neonatal-Health-Prediction-ML-
# Machine Learning Prediction Web App

This project is a Flask-based web application that predicts outcomes using Machine Learning models.

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML / CSS
- Random Forest
- Decision Tree

## Features
- Web interface for user input
- Machine learning prediction
- Random Forest and Decision Tree models
- Real-time prediction results

## Project Structure
app.py – Main Flask application  
templates/ – HTML frontend  
random_forest_model.pkl – Trained ML model  
decision_tree_model.pkl – Trained ML model  

## How to Run

pip install flask scikit-learn pandas numpy  
python app.py

Then open:
http://127.0.0.1:5000
